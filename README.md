## Proyecto_Sprint6
Datos del proyecto del Sprint 6 del Bootcamp de ciencia de datos de Tripleten

En esta oportunidad se ha realizado un análisis de los datos de consumo de 500 usuarios de la compañía telefónica Megaline en 2018. Se realizó una revisión inicial y ajuste de los datos para un mejor manejo de los mismos y tener un análisis más sólido.
